# Unity Project Inventory Tool

Unityプロジェクト内のVPM、UPM、Built-in Package、Editorツール、
Assets配下の内容を棚卸しするために作成した個人用Unity Editor拡張です。

自分のUnity／VRChatプロジェクト管理用に作成したものを、
参考・再利用目的で公開しています。
